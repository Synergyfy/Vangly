# COMPLETE ORGANIZATION ADMIN DASHBOARD REDESIGN PROMPT

Redesign and restructure the Organization Admin Dashboard for our outreach and engagement platform.

IMPORTANT:
This platform is MOBILE-FIRST.

Most users will use:
- mobile phones
- small screens
- low attention span workflows
- quick actions

The UI/UX must prioritize:
- mobile responsiveness first
- simplicity
- speed
- clean hierarchy
- progressive navigation
- modern SaaS feel
- easy usage for non-technical users

The platform is designed primarily for:
- churches
- ministries
- NGOs
- outreach organizations
- volunteer organizations
- communities

The core goal of the platform is:
- outreach tracking
- form-based data capture
- invite/follow-up systems
- centralized contact management
- distributed evangelism/outreach operations

IMPORTANT:
This is NOT a CRM.
This is NOT project management software.
This is NOT community social media.

Keep the experience lightweight, clean, and outreach-focused.

---

# 🔷 MAIN ORGANIZATION ADMIN NAVIGATION

The Organization Admin Dashboard should ONLY contain these top-level tabs:

1. Overview
2. Locations
3. Messaging
4. Brand Identity
5. Wallet
6. Settings

DO NOT add:
- Users tab
- Groups tab
- Forms tab
- Contacts tab
- Teams tab

Those should exist contextually inside locations.
